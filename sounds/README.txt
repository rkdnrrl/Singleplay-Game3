사운드 폴더 안내
================

폴더
----
  sounds/bgm/   … 배경음 (보통 길고 루프)
  sounds/sfx/   … 효과음 (짧은 클립)

파일 형식
---------
  MP3 또는 OGG 권장 (짧은 효과음은 WebM도 가능).
  브라우저마다 선호 포맷이 다를 수 있어, 나중에 코드에서 확장자만 바꿔도 되도록
  아래 "권장 파일명"을 그대로 쓰는 것을 추천합니다.

BGM (sounds/bgm/) — 권장 파일명
--------------------------------
  main_loop.mp3       … 게임 메인(대기~플레이 공통) 루프
  minigame_loop.mp3   … 미니게임 전용 루프 (쓰지 않으면 main만 두어도 됨)

  → 지금은 파일만 넣어 두면 되고, 재생은 아직 game.js에 연결하지 않은 상태일 수 있습니다.

효과음 (sounds/sfx/) — 권장 파일명 (코드에서 맞추기 쉬운 영문+스네이크 케이스)
-----------------------------------------------------------------------------
  ui_click.mp3           … 버튼/UI 탭
  cast_start.mp3         … "아이템 회수 시작" / 캐스트
  minigame_tick.mp3      … 미니게임 바 움직임(선택, 잦으면 생략 가능)
  minigame_success.mp3  … 미니게임 성공
  minigame_fail.mp3     … 미니게임 실패
  catch_reveal.mp3      … 결과 카드/낚시 성공 연출
  coin_gain.mp3         … 코인 획득·표시
  sell_complete.mp3     … 판매 완료
  scan_pulse.mp3        … 스캔 UI(선택)
  inv_open.mp3          … 적재함 확대 보기 열기(선택)

  사용하지 않는 이름의 파일은 넣지 않아도 됩니다.
  같은 역할에 다른 이름을 쓰고 싶으면, 나중에 sound 연동할 때 경로만 맞추면 됩니다.

주의
----
  모바일 브라우저는 첫 소리 재생 전에 사용자 탭/클릭이 있어야 하는 경우가 많습니다.
  (첫 버튼 누른 뒤부터 효과음이 나오게 하는 식으로 구현하는 것이 안전합니다.)
